# Portafolio Albert

Portafolio personal minimalista con React, Vite y Tailwind CSS.

## Proyectos incluidos

- **JJ Logistic** - Página web corporativa
- **SafePet** - Sitio web para cuidado de mascotas
- **Inversiones Duvan** - Página web institucional
- **Inversiones Duvan** - Catálogo

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en el navegador.

## Build

```bash
npm run build
```

Los archivos se generan en la carpeta `dist/`.

## Personalización

- **URLs de proyectos:** Edita `src/data/projects.js` con los enlaces reales
- **Imágenes:** Sustituye las URLs de placeholder por capturas de tus proyectos
- **Contacto:** Actualiza el email y enlaces sociales en `src/components/Contact.jsx`
- **Sobre mí:** Modifica el texto en `src/components/About.jsx`

## Deploy

Puedes desplegar en Vercel, Netlify o GitHub Pages de forma gratuita.
